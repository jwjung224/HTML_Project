Portfolio Style Guide

body { background: url(star.jpg); padding: 10px;} - This style embedding picture from url assigned and padding clears an area around the content by 25 pixel in all page.

div { background-color: rgba(70,70,30, 0.6); color: white;} - This style sets font color white and background color as RGB 70,70,70 with 60% of transparent in Division.

h1 { background-color: yellow; text-transform: uppercase; color: navy;} - This style sets background color as yellow and font color as navy, and all letter change to upper-case in Heading 1.

h2 { font-size: x-large; text-transform: lowercase; text-decoration: underline;} - This style sets font size x-large with underline. Also, it changes all letters to lower-case in Heading 2.

h3 { background-color: aqua; color: red;} - This style sets background aqua color and font red color in Heading 3.

p { color: white;} - This sets all paragraph font to white color.

ul { color: white;} - This sets all list font to white color.

img { border: 5px solid #008000;} - This sets image surround with line of border with 5 pixel think and green color when I embed image.

small{ background-color: black; color: white;} - This sets background color to black and font color to white in small.

.value, .rule { font-family:"Courier New", Courier, monospace;} - This class sets font sytle to Courier with monospace.

.underline{ text-decoration: underline;} This class sets underline.

.frame { font-family: arial, helvetica, sans-serif; font-size: 90%; border: 7px solid yellow; line-height: 8; color: white;} - This class sets font type to arial, helvetica or sans-serif with size of 90% from normal font size. Also, it make border with yellow 7 pixels, height of line become 8, and font color is white.
