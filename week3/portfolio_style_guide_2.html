Portfolio Style Guide

Overall Page Fomatting

  • font size is 12 point in general. background embedded dog's footprint image. Font set by comic sans ms at first if any web browser can't recognize comic then, it uses arial. If arial also does not work either then, it uses sans-serif font.

body { font: 12pt comic sans, arial, sans-seif; background: url(../images/footprint.jpg); }

Header Section

  • h1 in header makes background color as yellow and text aline at center. Also, all text Capitalized with navy color.

header h1 { background-color: yellow; text-align: center; text-transform: uppercase; color: navy;}

  • h2 in header makes all text first character is little bigger than other character in one word. Also, all texts are underlined.

header h2 { font-variant: small-caps; text-decoration: underline; }

Nav Section

  • Unorderd list in nav set font as bold and background color as RGB 70,70,30 with 35% of transparent. Also, background margin both left and right from 50 points and space between line is two times of parent's font size.

nav ul { font-weight: bold; background-color: rgba(70,70,30, 0.35); margin-left: 50pt; margin-right: 50pt; line-height: 2em; }

Content Section

I did not create specific Content ID but these selectors are what I use it for content.

  • p use navy color. font is Arial first, if not working then use helvetic, if both prior fonts not working then, use sans-serif font.

p { font-family: arial, helvetica, sans-serif; color: navy; }

  • ul use font is comic first, if not working then use arial, if both prior fonts not working then, use sans-serif font. This has size 12 points

ul { font: 12pt comic sans, arial, sans-seif; color: navy; }

  • Embedding image surround with line of border with 120% thick and green color.

img { border: 1.2em solid #008000 }

  • makes font color as green with 120% size of small's parent's font.

small{ font: 1.2em; color: green; }

Footer Section

  • Footer makes border silver color with thick line.

footer { border: solid thick silver; }

Class Section

  • Frame class sets font as arial first. Use helvetica if not available on browser at second. If both prior options not working then, use san-serif font. Frame class also font color is green, size 80% of its parent's font size and space between line is 200% of its parent's font size.

.frame { font-family: arial, helvetica, sans-serif; font-size: .8em; line-height: 2em; color: green; }

  • Underline class sets all text underlined.

.underline{ text-decoration: underline; }
