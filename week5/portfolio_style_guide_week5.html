Overall Page Fomatting

font size is 12 point in general. background embedded custom blue image. Font set by comic sans at first. It uses comic sans ms if any web browser can't recognize comic sans. If neither comic sans nor comic sans ms does not work then, it uses sans-serif font. If there is no background image, background color would be #483D8B (darkslateblue). Page font color is lime unless it specify color.

body { font: 12pt comic sans, comic sans ms, sans-seif; background: url(../images/blue.jpg); background-color: #483D8B; color: lime; }

This makes default link color as snow.

a:link { color: snow; }

This makes link that visited once color to skyblue.

a:visited { color: skyblue; } V

a:hover makes link that I am placing over color as pink.

a:hover { color: pink; }

a:active makes link that I am clicking on color as orange.

a:active{ color: orange; }

Header Section

h1 in header makes background color as snow and text aline at center. Also, word start letter is bit bigger than rest of word. Font color is navy.

header h1 { background-color: snow; text-align: center; color: navy; font-variant: small-caps; }

h2 in header makes all text first character is little bigger than other character in one word. Also, all texts are underlined.

header h2 { font-variant: small-caps; text-decoration: underline; }

Nav Section

  • Unorderd list in nav set font as bold and background color as RGB 70,70,30 with 50% of transparent. Padding is 200% of its parent font size. Space between line is 175% of parent's font size. Also, circle sign in front of every sentence removed by list-style-type as none.

nav ul { font-weight: bold; background-color: rgba(70,70,30, 0.50); padding: 2em; line-height: 1.75em; list-style-type: none; }

Content Section

I did not create specific Content ID but these selectors are what I use it for content.

  • Embedding image surround with line of border with 120% thick and green color.

img { border: 1.2em solid #008000 }

  • makes font size as 120% size of small's parent's font with color #ADFF2F(greenyello).

small{ font: 1.2em; color: #ADFF2F;}

Footer Section

There is no particular set up for footer section

footer { border: solid thick silver; }

Class Section

rememberContext class sets font as verdana first. Use arial if verdana is not available on browser. If both prior options not working then, use san-serif font. All characters are bold with font color seashell. Also, font size is 150% of its parent font size.

.rememberContext{ font-family: verdana, arial, Helvetica, sans-serif; font-weight: bold; font-size: 1.5em; color: seashell; }

Frame class sets font as Lucida Console first. Use courier if not available on browser at second. If both prior options not working then, use monospace font. Frame class also has size 80% of its parent's font size and space between line is 200% of its parent's font size.

.frame { font-family: "Lucida Console", courier, monospace; font-size: .8em; line-height: 2em; }

headerName class sets font as verdana first. If verdana does not work then, set as arial. If both first and second is not working then, helvetica. If the prior three font does not work, then using sans-serif font. Also this class has border at the bottom with silver color with think line.

.headerName { font-family: verdana, arial, helvetica, sans-serif; border-bottom: solid thick silver; }

titleSign class sets font as verdana first. If verdana does not work then, set as arial. If both first and second is not working then, helvetica. If the prior three font does not work, then using sans-serif font. All font comes with bold and align at center. with #F0FFF0(honeydew) color.

.titleSign { font-family: verdana, arial, helvetica, sans-serif; font-weight: bold; text-align: center; color: #F0FFF0; }
